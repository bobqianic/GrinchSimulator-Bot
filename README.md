## GrinchSimulator-Bot

GrinchSimulator-Bot is a client-side Minecraft/Fabric educational project that demonstrates block scanning, custom rendering overlays, pathfinding, and automated movement logic in a sandbox environment.

The project detects specific player-head blocks (grinch), highlights them visually through an X-ray-style overlay, calculates paths toward reachable targets, and optionally assists movement along those paths. It is intended as a learning example for working with Minecraft client events, world rendering, block/entity inspection, collision checking, asynchronous path computation, and modular Java code organization.

This project is provided for educational and research purposes only. It is meant to help developers understand client-side rendering, pathfinding algorithms, game-world analysis, and Fabric mod structure. It should not be used to gain unfair advantages on public or multiplayer servers, violate server rules, or disrupt other players’ experiences.
